# Task Manager Python

Acesta este un mini-proiect în Python: un Task Manager simplu care permite utilizatorului să adauge, vizualizeze, șteargă și marcheze task-uri ca făcute.

## Funcționalități

- Adaugă task-uri noi
- Afișează toate task-urile
- Șterge un task existent
- Marchează un task ca făcut
- Meniu interactiv în terminal
